# postgres-pool

Set up a connection pool for a Postgres server.

## Usage

    (postgres-pool {:host "localhost" :port 5432
                    :username "foo" :password "bar"
                    :database "mydb"})

## Installation

Available at Clojars, add this to `project.clj` and use `lein`:

    [postgres-pool "1.0.0"]
